✨ Tips Cepat Hutan
bit terakhir 1 = ganjil bit terakhir 0 = genap SHL = ×2 SHR = ÷2 4 bit = 1 nibble 8 bit = 1 byte NOT = flip semua bit 0xFF = 255
Hafalan Cepat: Pangkat 2 selalu dimulai dengan 1 dan diikuti 0-0-0… Jadi 2⁵ = 100000. Jumlah digit biner = eksponen + 1. Mudah kan?
Sign Bit: Bit paling kiri jadi tanda dalam sistem signed. 0 = positif, 1 = negatif. Contoh: 10000001 = -127 (two's complement).
Two's Complement: Cara komputer simpan negatif. Flip semua bit (NOT) lalu tambah 1. Contoh: -5 = NOT(00000101) + 1 = 11111011
